# 우리집 등원 도우미 🏫

아이 둘이 부모 없이 아침 등원 루틴을 스스로 소화하도록 돕는 시스템.
세 부분으로 구성: **① 알람+체크리스트 웹앱** (이 저장소) · **② 음성통화** (카카오톡) · **③ 현관 홈캠** (iPhone 11 + Alfred).

---

## ① 알람 + 체크리스트 웹앱

### 동작 방식
- 집 패드에 이 웹앱을 항상 띄워둔다. 시간표(기본: 07:00 기상 → 08:32 등교)에 맞춰 **알람 소리 + 음성 안내**가 나온다.
- 아이가 할 일을 끝내면 자기 이름 버튼을 탭 → **아빠 폰에 즉시 푸시 알림**.
- 아빠는 부모 모드 링크로 회사에서 **실시간 체크 현황**을 볼 수 있다.
- 🚨 "아빠 부르기" 버튼: 긴급 시 아빠 폰이 최우선 알림으로 울린다.
- 미완료 알림: 08:35(설정 가능)까지 안 끝난 항목이 있으면 아빠에게 자동 푸시.
- 시간표는 설정(⚙️)에서 자유롭게 수정 — 내년 3월 출근 시간이 바뀌어도 코드 수정 불필요.

### 배포 (GitHub Pages, 1회)
```bash
cd ~/projects/HomeCamera
git init && git add -A && git commit -m "등원 도우미 초기 버전"
gh repo create dungwon-helper --public --source=. --push
gh api repos/{owner}/dungwon-helper/pages -X POST -f 'source[branch]=main' -f 'source[path]=/'
# 몇 분 뒤 https://<계정명>.github.io/dungwon-helper/ 에서 접속 가능
```

### 기기 설정
1. **가족 채널 이름 정하기**: 가족만 아는 임의 문자열 (예: `dungwon-kim-7532`).
   ⚠️ 이 이름이 곧 비밀번호다 — 추측 가능한 이름(`kids`, `home`)은 쓰지 말 것.
2. **집 패드** (아이 모드): 브라우저로 배포 주소 접속 → 첫 화면 탭 → 설정에서 채널 이름·아이 이름 입력 → 저장.
   - 홈 화면에 추가(전체화면): Safari 공유 → "홈 화면에 추가".
   - **자동 잠금 끄기**: iPad 설정 → 디스플레이 및 밝기 → 자동 잠금 → 안 함. (웹앱이 화면 깨우기를 시도하지만 기기 설정이 확실하다)
   - 아침마다 화면을 한 번 탭해야 소리가 활성화된다 (iPad Safari 정책).
3. **아빠 폰 (S25)**:
   - [ntfy 앱](https://play.google.com/store/apps/details?id=io.heckel.ntfy) 설치 (무료, 광고 없음) → 구독 추가 → 채널 이름 입력 → 체크·긴급호출 푸시 수신.
   - 실시간 현황: 설정 화면에 표시되는 "부모 모드" 링크를 크롬 북마크/홈 화면에 추가.
4. **엄마 폰 (iPhone 13)**: App Store 의 ntfy 앱 + 부모 모드 링크, 동일하게.

### 연결 테스트
설정 모달의 **[테스트 푸시]** 버튼 → 아빠 폰 ntfy 앱에 알림이 오면 성공.

## ② 위급 시 음성통화 (구매·개발 불필요)

- 집 **iPad에 카카오톡 iPad 버전** 설치 (App Store) — 아내 계정으로 로그인하거나 아이 계정 생성.
- 즐겨찾기에 **아빠·엄마만** 등록 → 아이는 프로필 탭 → 통화 버튼 → **보이스톡**.
- 설치 후 실제 보이스톡 1회 테스트 필수 (iPad 마이크 권한 허용 확인).
- 보조 수단: Alfred 홈캠의 **양방향 오디오** — 아빠가 S25 Alfred 앱에서 현관 카메라(iPhone 11)의 스피커로 말할 수 있다 (워키토키).
- 웹앱의 🚨 버튼은 "아빠에게 신호" 용도 — 신호를 받으면 아빠가 카톡 보이스톡을 걸어주는 흐름.

## ③ 현관 홈캠 (iPhone 11 재활용, 0원)

1. **iPhone 11**: App Store 에서 **AlfredCamera** 설치 → "카메라" 역할 선택.
2. **S25**: Play 스토어에서 AlfredCamera 설치 → "뷰어" 역할 → 같은 계정(구글 계정 연동) 로그인 → 자동 페어링.
3. iPhone 11 을 현관이 보이는 위치에 거치, **충전기 상시 연결** (배터리로만 두면 오전 중 방전됨).
4. 무료 기능: 실시간 시청, 움직임 감지 알림, 양방향 오디오. (Premium 은 화질·녹화 확장 — 일단 무료로 충분)
5. 설정 팁: 움직임 감지 알림은 등교 시간대(08:20~08:45)에만 유용 — 상시 알림이 시끄러우면 감지 구역/일정 조정.
