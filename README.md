# 우리집 등원 도우미 🏫

아이 둘이 부모 없이 아침 등원 루틴을 스스로 소화하도록 돕는 시스템.
세 부분으로 구성: **① 알람+체크리스트 웹앱** (이 저장소) · **② 음성통화** (iPhone 11 + 카카오톡) · **③ 현관 홈캠** (Tapo 카메라).

---

## ① 알람 + 체크리스트 웹앱

### 동작 방식
- 집 패드에 이 웹앱을 항상 띄워둔다. 시간표(기본: 07:00 기상 → 08:32 등교)에 맞춰 **알람 소리 + 음성 안내**가 나온다.
- 아이가 할 일을 끝내면 자기 이름 버튼을 탭 → **아빠 폰에 즉시 푸시 알림**.
- 아빠는 부모 모드 링크로 회사에서 **실시간 체크 현황**을 볼 수 있다.
- 🚨 "아빠 부르기" 버튼: 긴급 시 아빠 폰이 최우선 알림으로 울린다.
- 미완료 알림: 08:35(설정 가능)까지 안 끝난 항목이 있으면 아빠에게 자동 푸시.
- 시간표는 설정(⚙️)에서 자유롭게 수정 — 내년 3월 출근 시간이 바뀌어도 코드 수정 불필요.

### 배포 (GitHub Pages, 1회)
```bash
cd ~/projects/HomeCamera
git init && git add -A && git commit -m "등원 도우미 초기 버전"
gh repo create dungwon-helper --public --source=. --push
gh api repos/{owner}/dungwon-helper/pages -X POST -f 'source[branch]=main' -f 'source[path]=/'
# 몇 분 뒤 https://<계정명>.github.io/dungwon-helper/ 에서 접속 가능
```

### 기기 설정
1. **가족 채널 이름 정하기**: 가족만 아는 임의 문자열 (예: `dungwon-kim-7532`).
   ⚠️ 이 이름이 곧 비밀번호다 — 추측 가능한 이름(`kids`, `home`)은 쓰지 말 것.
2. **집 패드** (아이 모드): 브라우저로 배포 주소 접속 → 첫 화면 탭 → 설정에서 채널 이름·아이 이름 입력 → 저장.
   - 홈 화면에 추가(전체화면): Safari 공유 → "홈 화면에 추가".
   - **자동 잠금 끄기**: iPad 설정 → 디스플레이 및 밝기 → 자동 잠금 → 안 함. (웹앱이 화면 깨우기를 시도하지만 기기 설정이 확실하다)
   - 아침마다 화면을 한 번 탭해야 소리가 활성화된다 (iPad Safari 정책).
3. **아빠 폰 (S25)**:
   - [ntfy 앱](https://play.google.com/store/apps/details?id=io.heckel.ntfy) 설치 (무료, 광고 없음) → 구독 추가 → 채널 이름 입력 → 체크·긴급호출 푸시 수신.
   - 실시간 현황: 설정 화면에 표시되는 "부모 모드" 링크를 크롬 북마크/홈 화면에 추가.
4. **엄마 폰 (iPhone 13)**: App Store 의 ntfy 앱 + 부모 모드 링크, 동일하게.

### 연결 테스트
설정 모달의 **[테스트 푸시]** 버튼 → 아빠 폰 ntfy 앱에 알림이 오면 성공.

## ② 위급 시 음성통화 — iPhone 11 = 아이들 전용 전화기

홈캠을 Tapo 전용 기기가 맡으면서 iPhone 11 이 자유로워짐 → **아이들 전용 통화 기기**로 사용.

- **iPhone 11**: 카카오톡 설치 (아이 계정 또는 아내 계정으로 로그인) → 즐겨찾기에 **아빠·엄마만** 등록.
  아이는 프로필 탭 → 통화 버튼 → **보이스톡**. 타자 불필요.
- 거실 등 아이 손 닿는 고정 위치에 두고 상시 충전. 다른 앱은 스크린타임으로 잠가서 "전화기" 역할만.
- 설치 후 실제 보이스톡 1회 테스트 필수 (마이크 권한 허용 확인).
- 백업 수단: 집 iPad 에도 카카오톡 iPad 버전 설치 가능 (동일 요령).
- 보조 수단: Tapo 카메라의 **양방향 오디오** — 아빠가 S25 Tapo 앱에서 현관 카메라 스피커로 말할 수 있다 (워키토키).
- 웹앱의 🚨 버튼은 "아빠에게 신호" 용도 — 신호를 받으면 아빠가 보이스톡을 걸어주는 흐름.

## ③ 현관 홈캠 (Tapo 카메라)

### 설치
1. **S25**: Play 스토어에서 **Tapo** 앱 설치 → TP-Link 계정 생성 → 카메라 추가 (앱이 QR/블루투스로 안내).
   Wi-Fi 는 **2.4GHz** 대역에 연결 (5GHz 만 잡히면 공유기에서 2.4GHz SSID 확인).
2. 현관 신발장 위 등 현관문이 보이는 위치에 거치 (스탠드 동봉, 필요 시 벽 브래킷) + 전원 상시 연결.
3. **microSD 카드** 삽입 → 앱에서 포맷 → 연속(또는 이벤트) 녹화 켜기 — "몇 시에 나갔지?" 되감기 가능.
4. **엄마 iPhone 13**: App Store 의 Tapo 앱 → 같은 계정 로그인 (또는 기기 공유 기능).
5. 외부(회사)에서도 앱 열면 바로 실시간 시청 — 별도 설정 불필요.

### 보안 3수칙 (구매 전 검토에서 확인한 취약점 이력 대응 — 반드시 적용)
1. 카메라 **펌웨어 자동 업데이트** 켜기 (앱 → 기기 설정 → 펌웨어).
2. TP-Link 계정에 **2단계 인증** 켜기.
3. 녹화는 클라우드 구독 대신 **microSD 로컬 저장** 사용.

### 운영 팁
- 움직임 감지 알림은 등교 시간대(08:20~08:45)에만 켜두면 소음 없이 유용 (앱의 알림 일정 기능).
- 프라이버시 모드: 가족이 다 집에 있는 주말엔 렌즈를 물리적으로 닫아둘 수 있다.

### 추가 구매 (소액)
| 품목 | 용도 | 예상가 | 어디서 |
|---|---|---|---|
| microSD 32~128GB | 로컬 녹화 | ~1만원 | 쿠팡 (삼성/샌디스크 정품) |
| 라이트닝 케이블 + 어댑터 | iPhone 11 상시 충전 | ~1만원 | 쿠팡 |
